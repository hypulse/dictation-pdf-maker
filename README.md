# Dictation PDF Generator

`txt` 또는 `srt` 파일을 읽거나, 화면에서 제목과 본문을 직접 입력해서 dictation용 빈칸 PDF를 만드는 아주 단순한 Streamlit 앱입니다.

## 기능

- `txt`, `srt` 업로드
- 제목 + 본문 직접 입력
- 직접 입력 시 `일반 텍스트`, `SRT` 선택
- `일반 텍스트` 입력 시 `텍스트 패턴` 선택
- `srt` 입력일 때 시간 표기 옵션 제공
- `단어 단위 밑줄`
- `문장 단위 밑줄`
- `각 문장의 첫 단어 표기`
- 결과 미리보기
- 문제 PDF 다운로드
- 답지 PDF 다운로드

## 실행

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## 처리 기준

- 형태소 분석이나 토큰화는 하지 않습니다.
- `srt` 파일은 블록 단위 문단을 유지한 채 타임코드를 추출하고, 원하면 시작 시각만 `HH:MM:SS` 형태로 함께 출력할 수 있습니다.
- `텍스트 패턴`을 `PodScripts Transcript`로 고르면 `Starting point is HH:MM:SS` 형식을 시간 표기처럼 처리합니다.
- `단어 단위 밑줄`은 공백 기준으로 단어를 구분해 공백은 유지하고, PDF에서는 각 단어를 `_` 대신 길이에 비례한 하나의 수평선으로 그립니다. 현재 기본값은 `글자 수 x 6px`, 최소 `18px`입니다.
- `문장 단위 밑줄`은 문장 전체를 하나의 연속된 숨김 영역으로 보고, 공백 길이도 포함해서 PDF에서 실제 수평선으로 그립니다.
- `각 문장의 첫 단어 표기`를 켜면 문장 앞의 `-`, 따옴표, 괄호 같은 기호만 있는 토큰은 건너뛰고, 첫 의미 있는 단어만 남긴 뒤 나머지는 선택한 방식대로 밑줄 처리합니다.
- 문장 끝의 마침표, 물음표, 느낌표 같은 문장부호는 문제지에서도 그대로 보이도록 유지합니다.
- 답지 PDF는 원문 문장을 그대로 출력합니다.
