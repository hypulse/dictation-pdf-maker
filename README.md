# Dictation PDF Maker

영어 자막, 팟캐스트 스크립트, 듣기 대본으로 바로 딕테이션 학습지를 만들기 위한 간단한 도구입니다.

`.txt` 또는 `.srt` 파일을 넣거나, 텍스트를 직접 붙여 넣으면:

- 빈칸이 들어간 문제 PDF
- 원문이 그대로 들어간 답지 PDF

를 바로 내려받을 수 있습니다.

유튜브 영상 설명란의 GitHub 링크를 통해 들어오신 분들은, 아래 사용 방법만 따라 실행하면 됩니다.

## 이런 용도로 만들었습니다

- 영어 유튜브 자막으로 딕테이션 자료 만들기
- 팟캐스트 transcript로 듣기 연습지 만들기
- 수업용/개인 공부용 빈칸 문제 빠르게 만들기
- 원문 편집 없이 바로 PDF로 정리하기

## 주요 기능

- `.txt`, `.srt` 파일 업로드
- 제목 + 본문 직접 입력
- `단어 단위 밑줄`
- `문장 단위 밑줄`
- `각 문장의 첫 단어 표기`
- 시간 정보가 있으면 timestamp 표시 가능
- 변환 결과 미리보기
- 문제 PDF / 답지 PDF 각각 다운로드

## 실행 방법

### 가장 간단한 실행 방법

Python과 `pip`가 설치되어 있다면, 가상환경 없이도 바로 실행할 수 있습니다.

```bash
python3 -m pip install -r requirements.txt
```

```bash
python3 -m streamlit run app.py
```

실행 후 브라우저가 열리면 바로 사용할 수 있습니다.

### 가상환경을 사용해서 실행하는 방법

패키지 충돌을 피하고 싶다면 가상환경을 만들어 실행해도 됩니다.

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
python3 -m streamlit run app.py
```

## 사용 방법

1. 앱을 실행합니다.
2. 입력 방식을 고릅니다.
   - `파일 업로드`: `.txt` 또는 `.srt` 파일 사용
   - `직접 입력`: 제목과 본문을 직접 붙여 넣기
3. 밑줄 방식을 고릅니다.
   - `단어 단위 밑줄`: 단어마다 따로 빈칸 처리
   - `문장 단위 밑줄`: 문장 전체를 길게 빈칸 처리
4. 필요하면 `각 문장의 첫 단어 표기`를 켭니다.
5. 시간 정보가 있는 자료라면 `시간 표기` 여부를 고릅니다.
6. 아래 미리보기에서 결과를 확인합니다.
7. `문제 PDF 다운로드`, `답지 PDF 다운로드` 버튼으로 파일을 받습니다.

## 입력 형식

### `srt`

- 자막 블록 단위로 읽습니다.
- 시간 정보가 있으면 시작 시각을 `HH:MM:SS` 형태로 표시할 수 있습니다.

### 일반 텍스트

- 줄 단위 또는 문단 단위로 처리합니다.
- 직접 입력 시 `텍스트 패턴`을 고를 수 있습니다.
- 영어 공부용 팟캐스트 transcript를 직접 붙여 넣어서 바로 PDF로 만들 수 있습니다.

### `PodScripts Transcript`

- `Starting point is HH:MM:SS` 형식을 시간 정보처럼 인식합니다.
- [PodScripts](https://podscripts.co) 에서 에피소드를 찾은 뒤 transcript를 복사해서 `직접 입력`에 붙여 넣으면 편하게 사용할 수 있습니다.

## 출력 파일

- `*_dictation.pdf`: 빈칸 문제지
- `*_answer_key.pdf`: 원문 답지

## 처리 방식

- 형태소 분석이나 복잡한 NLP 처리는 하지 않습니다.
- 띄어쓰기 기준으로 단순하게 처리합니다.
- 문장 끝 마침표, 물음표, 느낌표 같은 문장부호는 가능한 한 유지합니다.
